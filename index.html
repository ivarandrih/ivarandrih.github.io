About This Project

Built as part of a social data analysis and visualization course, this site brings together multiple formats for exploring the same dataset.

The goal of the project is not just to show charts, but to design a clear and readable experience for exploring San Francisco crime data. The site combines:

  • a static figure for quick overview,
  • an interactive Plotly chart for temporal exploration,
  • and a Folium map for geographic context.
